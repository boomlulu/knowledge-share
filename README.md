# 知识分享

这个仓库用于沉淀技术分享材料、演讲分镜和 H5 演讲播放器相关文件。

## 当前内容

- `帧同步技术分享/帧同步技术分享-大纲.md`：1 小时技术分享的大纲与章节草稿。
- `帧同步技术分享/H5演讲播放器-分镜.md`：H5 演讲播放器的演讲逻辑版分镜。
- `帧同步技术分享/H5分镜-审核总览.md`：用于碎片化审核分镜的轻量总览。
- `帧同步技术分享/h5/确定性雪崩-demo.html`：确定性雪崩演示页面。

## 原则

- 真实案例优先，不补虚构技术故事或数据。
- H5 服务于演讲逻辑，不为了页面数量拆分内容。
- 所有内容围绕：偏差、复杂度和信任的管理。
