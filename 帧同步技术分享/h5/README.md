# H5 演讲播放器

这里放帧同步技术分享的 H5 演讲播放器与演示页面。

当前已有：

- `index.html`：H5 演讲播放器骨架。
- `styles.css`：播放器样式。
- `app.js`：镜头数据、渲染逻辑和键盘交互。
- `UI约定.md`：播放器 UI、交互和协作约定。
- `references/视觉参考-20260625.png`：当前浅色未来极简风格参考图。
- `确定性雪崩-demo.html`：用于展示极小差异如何在连续模拟中被放大的互动演示。

## 打开方式

直接用浏览器打开 `index.html` 即可。排练时建议进入全屏，用方向键切换镜头。
